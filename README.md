# Python Learning

A beginner-friendly project for learning Python fundamentals.

## Getting Started

### Prerequisites

- Python 3.8 or higher

### Running the Hello World Script

```bash
python hello.py
```

Expected output:

```
Hello, World!
```

## Project Structure

```
python-learning/
├── hello.py       # Hello world script
└── README.md      # This file
```

## Next Steps

- Modify `hello.py` to greet different names
- Add new scripts as you learn more Python concepts
- Explore variables, loops, functions, and classes
